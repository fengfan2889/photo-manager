# 搜索过滤功能设计 - PhotoManager

## 功能概述

增强 Gallery 页面的搜索和过滤功能，支持多维度搜索和智能过滤，提供类似专业照片管理软件的搜索体验。

## 设计目标

1. **多字段搜索**：在多个照片属性字段中进行全文搜索
2. **精确过滤**：支持标签、日期范围、评分、人像等精确过滤
3. **组合搜索**：支持搜索词和过滤条件的组合使用
4. **用户友好**：提供直观的UI和智能提示
5. **性能优化**：确保在大量照片库下仍能快速响应

## 现有功能分析

### 后端现有能力 (`python/src/photo_repo.py`)

当前 `get_list` 方法支持：
- 基础分页：`limit`, `offset`
- 评分过滤：`rating` (0-5)
- 人像标记：`is_portrait`
- 标签过滤：`tag_id` (单标签)
- 日期范围：`date_from`, `date_to` (拍摄时间)
- 简单搜索：`search` (仅在 `file_name` 和 `camera_model` 字段中LIKE匹配)

### 前端现有能力 (`electron/src/renderer/src/pages/Gallery.tsx`)

- 搜索状态管理：`useState('')`
- 搜索触发：`handleSearch()` 函数
- 照片加载：通过 `window.electronAPI.pythonExec('query-photos', {...})`

#### 扩展搜索字段
在 `photo_repo.py` 的 `get_list` 方法中扩展搜索范围：

```python
if search:
    search_term = f'%{search}%'
    sql += """ AND (
        p.file_name LIKE ? 
        OR p.camera_model LIKE ?
        OR p.camera_make LIKE ?
        OR p.taken_at LIKE ?
        OR EXISTS (
            SELECT 1 FROM photo_photo_tag ppt 
            JOIN photo_tag t ON ppt.tag_id = t.id 
            WHERE ppt.photo_id = p.id AND t.name LIKE ?
        )
    )"""
    params.extend([search_term] * 5)  # file_name, camera_model, camera_make, taken_at, tag_name
```

#### 新增过滤参数
添加多标签支持、日期时间精度控制等：

```python
# 新增参数

```

#### 多标签过滤实现
```python
if tag_ids and len(tag_ids) > 0:
    placeholders = ','.join(['?' for _ in tag_ids])
    sql += f" AND EXISTS (SELECT 1 FROM photo_photo_tag pt WHERE pt.photo_id = p.id AND pt.tag_id IN ({placeholders}))"
    params.extend(tag_ids)
```

### 4. 性能优化策略

#### 数据库索引建议
确保以下字段有索引：
- `photo_info.taken_at` (日期范围查询)
- `photo_info.rating` (评分过滤)
- `photo_info.file_name` (文件名搜索)
- `photo_info.camera_make`, `photo_info.camera_model` (相机信息搜索)
- `photo_photo_tag.tag_id` (标签关联查询)
- `photo_tag.name` (标签名称搜索)

#### 查询优化
1. 使用 EXISTS 而非 IN 进行标签过滤，避免大量标签时的性能问题
2. 搜索词前置%匹配无法使用普通索引，考虑：
   - 全文搜索引擎（SQLite FTS5）
   - 或接受前置%匹配的性能影响（对于中等规模照片库可接受）

### 5. API设计

#### IPC命令：query-photos (增强版)
参数：
```json
{
  "limit": 100,
  "offset": 0,
  "rating": null,           // 精确评分过滤
  "rating_min": null,       // 最小评分过滤 (>=)
  "is_portrait": null,      // 人像标记
  "tag_ids": [],            // 多标签过滤 (ID数组)
  "date_from": null,        // 开始日期 (YYYY-MM-DD)
  "date_to": null,          // 结束日期 (YYYY-MM-DD)
  "search": null,           // 多字段搜索词
  "search_fields": null     // 限制搜索字段 (可选)
}
```

### 8. 用户场景示例

1. **快速搜索**：在搜索框输入 "北京" → 找到所有文件名、相机信息或标签包含"北京"的照片
2. **标签过滤**：选择"旅游"标签 → 只显示标记为旅游的照片
3. **日期范围**：选择"2026年4月" → 只显示4月份拍摄的照片
4. **组合使用**：搜索 "北京" + 选择"旅游"标签 + 选择"4星以上" → 找到高评分的北京旅游照片
5. **精确过滤**：选择"人像" + "评分>=4" → 找到高评分的人像照片
