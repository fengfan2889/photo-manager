# 开发计划 - PhotoManager

## 项目目标

基于 Electron + Python 开发轻量级照片管理软件：
- 桌面端（Electron 窗口）+ 浏览器双入口
- 图片自动整理（按拍摄时间）
- SQLite 存储 + JSON 导出
- 照片标注（星级、标签）
