## 表结构

### photo_info（照片信息）

照片的核心数据表。

```sql
CREATE TABLE photo_info (
    -- 公共字段
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    -- 文件信息
    file_path       TEXT NOT NULL UNIQUE,
    file_hash       TEXT NOT NULL,
    file_size       INTEGER,
    file_name       TEXT NOT NULL,
    
    -- 时间信息
    taken_at        DATETIME,
    taken_at_local  DATETIME,
    created_at_file DATETIME,
    modified_at_file DATETIME,
    
    -- 相机信息
    camera_make     TEXT,
    camera_model    TEXT,
    lens_model      TEXT,
    iso             INTEGER,
    aperture        REAL,
    shutter_speed   TEXT,
    focal_length    REAL,
    
    -- 位置信息
    latitude        REAL,
    longitude       REAL,
    
    -- 整理信息
    organized_path  TEXT,
    organized_date  TEXT,
    
    -- 标注信息
    is_portrait     INTEGER DEFAULT 0,
    rating          INTEGER DEFAULT 0,
    
    -- 缩略图
    thumb_path      TEXT,
    
    -- 完整 EXIF
    exif_json       TEXT
);

-- 索引
CREATE UNIQUE INDEX uq_photo_info_hash ON photo_info(file_hash);
CREATE INDEX idx_photo_info_taken_at ON photo_info(taken_at);
CREATE INDEX idx_photo_info_organized_date ON photo_info(organized_date);
CREATE INDEX idx_photo_info_rating ON photo_info(rating);
CREATE INDEX idx_photo_info_is_portrait ON photo_info(is_portrait);
```

---

### photo_tag（标签）

用户自定义标签。

```sql
CREATE TABLE photo_tag (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    name        TEXT NOT NULL UNIQUE,
    color       TEXT DEFAULT '#808080',
    parent_id   INTEGER REFERENCES photo_tag(id) ON DELETE SET NULL
);

CREATE UNIQUE INDEX uq_photo_tag_name ON photo_tag(name);
```

---

### photo_photo_tag（照片-标签关联）

照片与标签的多对多关系。

```sql
CREATE TABLE photo_photo_tag (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    photo_id    INTEGER NOT NULL REFERENCES photo_info(id) ON DELETE CASCADE,
    tag_id      INTEGER NOT NULL REFERENCES photo_tag(id) ON DELETE CASCADE,
    UNIQUE (photo_id, tag_id)
);

CREATE INDEX idx_photo_photo_tag_photo ON photo_photo_tag(photo_id);
CREATE INDEX idx_photo_photo_tag_tag ON photo_photo_tag(tag_id);
```

---

### photo_face_detect（人脸检测）

检测到的人脸记录。

```sql
CREATE TABLE photo_face_detect (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    photo_id        INTEGER NOT NULL REFERENCES photo_info(id) ON DELETE CASCADE,
    face_x          REAL,
    face_y          REAL,
    face_w          REAL,
    face_h          REAL,
    embedding       BLOB,
    subject_id      INTEGER REFERENCES photo_face_subject(id) ON DELETE SET NULL,
    confidence      REAL
);

CREATE INDEX idx_photo_face_detect_photo ON photo_face_detect(photo_id);
CREATE INDEX idx_photo_face_detect_subject ON photo_face_detect(subject_id);
```

---

### photo_face_subject（人物主题）

人物信息，用于人脸聚类和命名。

```sql
CREATE TABLE photo_face_subject (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    name            TEXT NOT NULL,
    avatar_path     TEXT,
    embedding_avg   BLOB,
    face_count      INTEGER DEFAULT 0
);

CREATE INDEX idx_photo_face_subject_name ON photo_face_subject(name);
```

### photo_log_operation（操作日志）

照片操作历史。

```sql
CREATE TABLE photo_log_operation (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    photo_id        INTEGER REFERENCES photo_info(id) ON DELETE SET NULL,
    action          TEXT NOT NULL,
    source_path     TEXT,
    dest_path       TEXT,
    action_type     TEXT,
    status          TEXT,
    error_msg       TEXT
);

CREATE INDEX idx_photo_log_operation_photo ON photo_log_operation(photo_id);
CREATE INDEX idx_photo_log_operation_status ON photo_log_operation(status);
```

## 附录：sys_setting 预设配置

| key | type | group_name | 默认值 | 说明 |
|-----|------|------------|--------|------|
| organize_mode | string | organize | copy | 整理模式 |
| organize_source | string | organize | (空) | 整理源目录 |
| organize_base | string | organize | (空) | 整理输出根目录 |
| organize_include_unknown | boolean | organize | true | 是否整理无法识别时间的照片 |
| time_priority | string | organize | exif>mtime>ctime | 时间优先级 |
| thumb_size | number | display | 200 | 缩略图大小 |
| grid_columns | number | display | 4 | 网格列数 |
| theme | string | display | light | 主题 |
| log_level | string | log | INFO | 日志级别 |
